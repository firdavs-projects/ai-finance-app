# BottomSheet Component

Нативноподобный компонент BottomSheet для мобильных устройств с поддержкой перетаскивания.

## Возможности

✅ **Плавное перетаскивание** - поддержка touch и mouse событий
✅ **Velocity tracking** - отслеживание скорости свайпа для быстрого закрытия
✅ **Умное поведение** - не конфликтует со скроллируемым контентом
✅ **Сопротивление** - естественное сопротивление при перетаскивании
✅ **Backdrop** - затемнение фона с blur эффектом
✅ **Анимации** - плавные анимации появления/исчезновения
✅ **Accessibility** - поддержка ARIA атрибутов

## Использование

```tsx
import { BottomSheet } from '@shared/ui/BottomSheet'

function MyComponent() {
  const [isOpen, setIsOpen] = useState(false)

  return (
    <>
      <button onClick={() => setIsOpen(true)}>
        Открыть
      </button>

      <BottomSheet
        isOpen={isOpen}
        onClose={() => setIsOpen(false)}
        title="Заголовок"
      >
        <div>Содержимое</div>
      </BottomSheet>
    </>
  )
}
```

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `isOpen` | `boolean` | ✅ | Контролирует видимость BottomSheet |
| `onClose` | `() => void` | ✅ | Callback при закрытии |
| `title` | `string` | ✅ | Заголовок в header |
| `children` | `ReactNode` | ✅ | Содержимое |

## Как работает

### Жесты закрытия

1. **Свайп вниз** - протяните панель вниз более чем на 150px
2. **Быстрый свайп** - быстро проведите вниз (velocity > 0.5 px/ms)
3. **Клик на backdrop** - нажмите на затемнённую область
4. **Кнопка закрытия** - иконка X в header

### Умное поведение

- Если контент прокручиваемый и не в начале, перетаскивание не начнётся
- При перетаскивании body блокируется от скролла
- Сопротивление делает движение более естественным
- Анимации используют cubic-bezier для нативного ощущения

## Стилизация

Компонент использует Tailwind CSS классы:
- `rounded-t-3xl` - скруглённые верхние углы
- `max-h-[80vh]` - максимальная высота контента
- `backdrop-blur-sm` - размытие фона

## Технические детали

- Построен на основе `@headlessui/react` Dialog
- Использует React hooks для управления состоянием
- Touch и mouse события для кросс-платформенности
- Предотвращает скролл body при открытом sheet
